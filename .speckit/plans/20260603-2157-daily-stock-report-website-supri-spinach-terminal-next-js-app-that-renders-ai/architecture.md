# Architecture: Daily stock report website — Supri Spinach Terminal. Next.js app that renders AI-generated markdown reports. Two sections: Report (daily 31-stock analysis) and Deep Research (per-ticker deep dive). Dynamic date-based routing from content/reports/ and content/deep-research/ directories. Dark trading terminal UI with sidebar navigation. Built with TypeScript, Tailwind v4, remark for markdown rendering.
